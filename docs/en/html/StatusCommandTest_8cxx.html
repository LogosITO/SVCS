Detailed Description

Integration tests for the StatusCommand class.

This file implements the test cases for the StatusCommandTest fixture. It verifies the command's ability to display the working tree status, covering the full repository status display and status checks for specific files. Tests include verifying correct reporting for staged, unstaged, and missing files, as well as handling the 'no repository' error condition.
