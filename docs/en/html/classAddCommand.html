Detailed Description

Implements the "add" command for the Simple Version Control System (SVCS).

This command is responsible for adding (staging) the specified files to the staging area of the repository for a subsequent commit.

The class inherits from ICommand and uses the Command pattern to execute its operations. It depends on ISubject (event bus) for user notifications and RepositoryManager for repository operations.
