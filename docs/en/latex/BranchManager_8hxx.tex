\doxysubsection{Detailed Description}
Declaration of the \doxylink{classBranchManager}{Branch\+Manager} class for handling version control branches. 

The \doxylink{classBranchManager}{Branch\+Manager} is responsible for all operations related to managing branches within the SVCS repository, including creation, deletion, renaming, and switching. It manages branch metadata in memory and persists changes to the underlying file system using utility methods for loading and saving state.
