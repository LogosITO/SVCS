\doxysubsection{Detailed Description}
Mock implementation of the Help\+Service interface for unit testing purposes. 

\begin{DoxyCopyright}{Copyright}
Copyright 2025 Logos\+ITO Licensed under MIT-\/\+License
\end{DoxyCopyright}
This class provides a mock implementation with the same interface as Help\+Service but with controlled behavior for testing. It includes utility methods ({\ttfamily set\+Available\+Commands}, {\ttfamily was\+Command\+Help\+Called}, etc.) to allow tests to configure the mock\textquotesingle{}s behavior and inspect its interactions. This mock is essential for unit testing the {\bfseries{Help\+Command}} without depending on the actual implementation of the CLI command registration system.
