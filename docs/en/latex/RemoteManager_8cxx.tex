This file implements the methods for the \doxylink{classRemoteManager}{Remote\+Manager} class. It handles the loading, saving, and in-\/memory management of remote repository configurations. Persistence is handled by reading from and writing to a dedicated \textquotesingle{}remotes\textquotesingle{} file (e.\+g., /path/to/repo/remotes) in a Git-\/config-\/like format. \begin{DoxyNote}{Note}
This implementation assumes the configuration is in a separate \textquotesingle{}remotes\textquotesingle{} file at the root of the repository, {\itshape not} inside {\ttfamily .svcs/config}. 
\end{DoxyNote}
