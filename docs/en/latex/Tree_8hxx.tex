Defines the Tree class, which represents the hierarchical structure (directory state) of a VCS repository.

The Tree object stores a list of Tree\+Entry structures, linking file names and modes to the hash IDs of Blob or other Tree objects. Its hash depends critically on the sorted content of these entries.
