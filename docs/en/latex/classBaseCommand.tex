\doxysubsection{Detailed Description}
Base class for all CLI commands. 

Provides a common infrastructure for interacting with the \doxylink{structEvent}{Event} Bus system (\doxylink{classISubject}{ISubject}) and utility methods for working with the repository. It is responsible for logging via the \doxylink{structEvent}{Event} Bus.
