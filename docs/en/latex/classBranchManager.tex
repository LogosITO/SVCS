\doxysubsection{Detailed Description}
Core service for managing version control branches (creation, deletion, switching). 

This class abstracts the details of branch storage and provides a high-\/level interface for branch manipulation. It uses an internal map to track branches and communicates status/errors via the \doxylink{classISubject}{ISubject} event bus.
