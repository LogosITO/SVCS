\doxysubsection{Detailed Description}
Factory for creating \doxylink{classICommand}{ICommand} objects by their string name. 

The \doxylink{classCommandFactory}{Command\+Factory} manages the lifecycle of commands, creating them dynamically using creator functions registered under unique names. Commands are constructed with shared ownership of an event bus (\doxylink{classISubject}{ISubject}) and a repository manager (\doxylink{classRepositoryManager}{Repository\+Manager}).
