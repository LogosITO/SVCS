\doxysubsection{Detailed Description}
Integration component for combining multiple command factories. 

Serves as a bridge between different command subsystems (CLI, \doxylink{classServer}{Server}). When creating a command, it tries each registered factory in order until one successfully creates the command. This design allows clean separation of concerns while providing a unified command interface to the application.
