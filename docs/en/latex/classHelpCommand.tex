\doxysubsection{Detailed Description}
Command for displaying help information about other commands. 

The {\bfseries{\doxylink{classHelpCommand}{Help\+Command}}} is the application\textquotesingle{}s built-\/in help utility. It can show a general list of all available commands or display detailed usage and descriptions for a specific command provided as an argument. It delegates the retrieval of command information to the {\bfseries{\doxylink{classHelpService}{Help\+Service}}} to maintain a clean separation of concerns.
