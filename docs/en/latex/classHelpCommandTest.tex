\doxysubsection{Detailed Description}
Google Test fixture for running unit/integration tests against the \doxylink{classHelpCommand}{Help\+Command}. 

This fixture is crucial for isolating the \doxylink{classHelpCommand}{Help\+Command}\textquotesingle{}s logic, ensuring it correctly delegates help requests to the {\bfseries{\doxylink{classMockHelpService}{Mock\+Help\+Service}}} and reports the resulting output via the {\bfseries{\doxylink{classMockSubject}{Mock\+Subject}}} (event bus).
