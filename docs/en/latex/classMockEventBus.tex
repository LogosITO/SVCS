Notifies all attached observers about an event. 

This method is made public so that component classes (like \doxylink{classObjectStorage}{Object\+Storage}) can publish events through the injected \doxylink{classISubject}{ISubject} pointer. The method is declared as {\ttfamily const} because event notification is a logically non-\/mutating action on the Subject\textquotesingle{}s state.
