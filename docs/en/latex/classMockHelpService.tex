\doxysubsection{Detailed Description}
A mock implementation of the \doxylink{classHelpService}{Help\+Service} for testing the \doxylink{classHelpCommand}{Help\+Command}. 

This mock allows test cases to\+:
\begin{DoxyEnumerate}
\item {\bfseries{Configure}} the list of available commands.
\item {\bfseries{Configure}} the descriptions for specific commands.
\item {\bfseries{Verify}} which command\textquotesingle{}s help was requested by the {\ttfamily \doxylink{classHelpCommand}{Help\+Command}}.
\item Use the provided \doxylink{classMockSubject}{Mock\+Subject} for emitting debug information during testing. 
\end{DoxyEnumerate}
