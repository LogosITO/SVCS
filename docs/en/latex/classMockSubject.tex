\doxysubsection{Detailed Description}
Mock implementation of ISubject (Event Bus). 

A mock implementation of ISubject for testing event-\/driven components.

Mock implementation of ISubject.


\begin{DoxyItemize}
\item This mock silences all notifications, ensuring that Index logic is tested without side effects from the event system.
\item This mock silences all event notifications, ensuring Object\+Storage tests run without side effects from the event system.

This mock allows test cases to\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item {\bfseries{Capture}} all events published via \doxylink{classMockSubject_a9aaffe9cd012979c90c186ea3ec86fc0}{notify()} method.
\item {\bfseries{Inspect}} captured events for specific types or messages.
\item {\bfseries{Clear}} the event history between test cases.
\item {\bfseries{Verify}} that specific events were published during execution. 
\end{DoxyEnumerate}
