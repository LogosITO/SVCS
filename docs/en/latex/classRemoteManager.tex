\doxysubsection{Detailed Description}
Manages the repository\textquotesingle{}s remote configurations. 

This class provides an interface to manage remote repositories. It handles adding, removing, and listing remotes, as well as persisting these configurations to and from the repository\textquotesingle{}s {\ttfamily .svcs/config} file.
