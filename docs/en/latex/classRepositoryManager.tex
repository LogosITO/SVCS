\doxysubsection{Detailed Description}
Manages all physical file and directory operations within the SVCS repository. 

The \doxylink{classRepositoryManager}{Repository\+Manager} is the core layer responsible for interacting directly with the file system for tasks like initialization, path management, staging, committing, and logging.
