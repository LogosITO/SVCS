\doxysubsection{Detailed Description}
Factory for creating server administration \doxylink{classICommand}{ICommand} objects. 

The \doxylink{classServerCommandFactory}{Server\+Command\+Factory} manages the lifecycle of server-\/specific commands, creating them dynamically using creator functions registered under unique names. Commands are constructed with shared ownership of an event bus (\doxylink{classISubject}{ISubject}) and a repository manager (\doxylink{classRepositoryManager}{Repository\+Manager}). This factory is specifically designed for server administration commands that require repository management.
