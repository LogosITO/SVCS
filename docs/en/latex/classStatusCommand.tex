\doxysubsection{Detailed Description}
Command for showing the current status of the repository. 

The \doxylink{classStatusCommand}{Status\+Command} displays\+:
\begin{DoxyItemize}
\item Current branch
\item Staged changes (ready to save)
\item Unstaged changes (not yet added)
\item Untracked files (new files)
\item Status of specific files when provided as arguments 
\end{DoxyItemize}
