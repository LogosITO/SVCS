\doxysubsection{Detailed Description}
Helper class for testing the non-\/abstract methods of \doxylink{classVcsObject}{Vcs\+Object}. 

This class implements all pure virtual methods of \doxylink{classVcsObject}{Vcs\+Object} with simple stubs, allowing its instances to be created for unit testing the hashing logic in \doxylink{classVcsObject_ae2b7544a63a2f0b1ef764dd4710a13e0}{Vcs\+Object\+::compute\+Hash}.
