\doxysubsection{Detailed Description}
Represents the state of a directory in the VCS, implementing the \doxylink{classVcsObject}{Vcs\+Object} contract. 

Stores a collection of \doxylink{structTreeEntry}{Tree\+Entry} objects, effectively mapping names to hash pointers.
