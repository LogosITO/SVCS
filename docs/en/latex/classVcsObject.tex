\doxysubsection{Detailed Description}
Abstract base class representing any storable, addressable object within the VCS. 

\doxylink{classVcsObject}{Vcs\+Object} enforces the mandatory presence of a hash ID, serialization logic, and type identification for all derived classes. It serves as the core domain entity for the object database.
