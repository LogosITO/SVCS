\doxysubsection{Detailed Description}
Google Test fixture for running unit/integration tests against the \doxylink{classVersionCommand}{Version\+Command}. 

This fixture provides the necessary infrastructure for testing the \doxylink{classVersionCommand}{Version\+Command}\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Initialize and clean up the mock event bus and the command object.
\item {\bfseries{Core Components\+:}} Instances of \doxylink{classMockSubject}{Mock\+Subject} (for capturing output) and the \doxylink{classVersionCommand}{Version\+Command} itself.
\item {\bfseries{Helper Methods\+:}} Simplify checking the captured event notifications for specific content. 
\end{DoxyItemize}
