\doxysubsection{Detailed Description}
Base class for all CLI commands. 

Provides a common infrastructure for interacting with the \doxylink{structsvcs_1_1services_1_1Event}{Event} Bus system (\doxylink{classsvcs_1_1services_1_1ISubject}{ISubject}) and utility methods for working with the repository. It is responsible for logging via the \doxylink{structsvcs_1_1services_1_1Event}{Event} Bus.
