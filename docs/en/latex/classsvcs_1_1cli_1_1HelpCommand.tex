\doxysubsection{Detailed Description}
Command for displaying help information about other commands. 

The {\bfseries{\doxylink{classsvcs_1_1cli_1_1HelpCommand}{Help\+Command}}} is the application\textquotesingle{}s built-\/in help utility. It can show a general list of all available commands or display detailed usage and descriptions for a specific command provided as an argument. It delegates the retrieval of command information to the {\bfseries{\doxylink{classsvcs_1_1cli_1_1HelpService}{Help\+Service}}} to maintain a clean separation of concerns.
