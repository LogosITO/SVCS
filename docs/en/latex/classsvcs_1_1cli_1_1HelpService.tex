\doxysubsection{Detailed Description}
Service that provides help information for commands. 

This service implements the application logic for displaying help. It uses a set of {\bfseries{callback functions}} ({\ttfamily std\+::function}) to retrieve command data (names, descriptions, usage) and display specific help. This design choice prevents a direct dependency on the \doxylink{classsvcs_1_1cli_1_1CommandFactory}{Command\+Factory}, thereby breaking a potential {\bfseries{circular dependency}}.
