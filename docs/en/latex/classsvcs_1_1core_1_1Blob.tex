\doxysubsection{Detailed Description}
Represents the content of a file (Binary Large Object). 

This is the lowest-\/level object in the VCS, storing only raw file bytes. The \doxylink{classsvcs_1_1core_1_1Blob}{Blob} object is immutable; its hash is solely dependent on its content.
