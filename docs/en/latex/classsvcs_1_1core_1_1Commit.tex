\doxysubsection{Detailed Description}
Represents a single, immutable historical point (snapshot) in the repository\textquotesingle{}s history. 

This object binds the content structure (\doxylink{classsvcs_1_1core_1_1Tree}{Tree} hash) with the metadata (author, message, parent history).
