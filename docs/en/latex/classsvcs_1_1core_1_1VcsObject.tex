\doxysubsection{Detailed Description}
Abstract base class representing any storable, addressable object within the VCS. 

\doxylink{classsvcs_1_1core_1_1VcsObject}{Vcs\+Object} enforces the mandatory presence of a hash ID, serialization logic, and type identification for all derived classes. It serves as the core domain entity for the object database.
