\doxysubsection{Detailed Description}
Abstract base class for all server administration commands. 

Inherits from \doxylink{classsvcs_1_1cli_1_1ICommand}{ICommand} and provides common infrastructure for server management operations including event notification and repository management. All server-\/specific commands should derive from this class.
