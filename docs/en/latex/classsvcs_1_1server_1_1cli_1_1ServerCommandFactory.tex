\doxysubsection{Detailed Description}
Factory for creating server administration \doxylink{classsvcs_1_1cli_1_1ICommand}{ICommand} objects. 

The \doxylink{classsvcs_1_1server_1_1cli_1_1ServerCommandFactory}{Server\+Command\+Factory} manages the lifecycle of server-\/specific commands, creating them dynamically using creator functions registered under unique names. Commands are constructed with shared ownership of an event bus (\doxylink{classsvcs_1_1services_1_1ISubject}{ISubject}) and a repository manager (\doxylink{classsvcs_1_1core_1_1RepositoryManager}{Repository\+Manager}). This factory is specifically designed for server administration commands that require repository management.
