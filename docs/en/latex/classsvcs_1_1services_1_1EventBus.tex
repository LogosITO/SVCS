\doxysubsection{Detailed Description}
A thread-\/safe implementation of \doxylink{classsvcs_1_1services_1_1ISubject}{ISubject} for centralized event distribution. 

This class uses std\+::weak\+\_\+ptr to store observers, preventing circular dependencies and ensuring automatic cleanup of destroyed observers. It inherits from std\+::enable\+\_\+shared\+\_\+from\+\_\+this to ensure that valid shared pointers to the bus can be created even inside its member functions.
