\doxysubsection{Detailed Description}
The Subject (Publisher) Interface (Abstract Base Class). 

Core classes (e.\+g., Repository, \doxylink{classsvcs_1_1services_1_1EventBus}{Event\+Bus}) implement this interface to allow observers to subscribe to events. The contract requires all management functions ({\ttfamily attach}, {\ttfamily detach}) to handle {\ttfamily std\+::shared\+\_\+ptr\texorpdfstring{$<$}{<}\doxylink{classsvcs_1_1services_1_1IObserver}{IObserver}\texorpdfstring{$>$}{>}}.
