\doxysubsection{Detailed Description}
A mock implementation of \doxylink{classsvcs_1_1services_1_1ISubject}{ISubject} for testing event-\/driven components. 

This mock allows test cases to\+:
\begin{DoxyEnumerate}
\item {\bfseries{Capture}} all events published via \doxylink{classsvcs_1_1test_1_1cli_1_1mocks_1_1MockSubject_abfc5bfa7fe007eca3e7e8dd8d84e1d9c}{notify()} method.
\item {\bfseries{Inspect}} captured events for specific types or messages.
\item {\bfseries{Clear}} the event history between test cases.
\item {\bfseries{Verify}} that specific events were published during execution.
