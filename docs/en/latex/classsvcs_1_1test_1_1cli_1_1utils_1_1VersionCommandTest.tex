\doxysubsection{Detailed Description}
Google Test fixture for running unit/integration tests against the \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1VersionCommand}{Version\+Command}. 

This fixture provides the necessary infrastructure for testing the \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1VersionCommand}{Version\+Command}\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Initialize and clean up the mock event bus and the command object.
\item {\bfseries{Core Components\+:}} Instances of \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1MockSubject}{Mock\+Subject} (for capturing output) and the \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1VersionCommand}{Version\+Command} itself.
\item {\bfseries{Helper Methods\+:}} Simplify checking the captured event notifications for specific content.
