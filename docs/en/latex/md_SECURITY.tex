We are committed to maintaining the security of our project. If you find a security issue, please help us by reporting it responsibly.

{\bfseries{Project Version\+: 0.\+1}}

\DoxyHorRuler{0}
\hypertarget{md_SECURITY_autotoc_md31}{}\doxysection{\texorpdfstring{How to Report a Vulnerability}{How to Report a Vulnerability}}\label{md_SECURITY_autotoc_md31}
Please do not post security issue details publicly (e.\+g., in a Git\+Hub Issue) until we have fixed the problem.

{\bfseries{Use this email to send your report\+:}} \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}artyukhov\href{mailto:.p@outlook.com}{\texttt{.\+p@outlook.\+com}}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

{\bfseries{What to include in your email\+:}}
\begin{DoxyEnumerate}
\item The name of the issue (e.\+g., "{}\+Login Bug,"{} "{}\+Crash"{}).
\item Clear steps to reproduce it (a step-\/by-\/step guide).
\item Which project version you were using (e.\+g., "{}0.\+1"{}).
\end{DoxyEnumerate}

\DoxyHorRuler{0}
\hypertarget{md_SECURITY_autotoc_md33}{}\doxysection{\texorpdfstring{Our Process}{Our Process}}\label{md_SECURITY_autotoc_md33}
We promise to handle your report quickly\+:


\begin{DoxyEnumerate}
\item {\bfseries{Acknowledgement\+:}} We will reply to your email within {\bfseries{2 days}} to confirm we received it.
\item {\bfseries{Fixing\+:}} We will start working on a fix as soon as possible.
\item {\bfseries{Thank You\+:}} Once the issue is fixed, we will thank you for your help.
\end{DoxyEnumerate}

We ask that you do not share details about the vulnerability for {\bfseries{90 days}} after your report to give us time to release and distribute the fix.
