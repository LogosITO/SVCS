\doxysubsection{Detailed Description}
Represents a single file entry in the staging area (\doxylink{classIndex}{Index}). 

An \doxylink{structIndexEntry}{Index\+Entry} stores information about a file that is staged for the next commit, including its object ID (hash) and file metadata used to detect modifications.
