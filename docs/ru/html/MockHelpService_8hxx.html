Подробное описание

Mock implementation of the HelpService interface for unit testing purposes.

This class inherits from HelpService and overrides its abstract methods (getAvailableCommands, getCommandDescription, showCommandHelp) with controlled implementations. It includes utility methods (setAvailableCommands, wasCommandHelpCalled, etc.) to allow tests to configure the mock's behavior and inspect its interactions. This mock is essential for unit testing the HelpCommand without depending on the actual implementation of the CLI command registration system.
