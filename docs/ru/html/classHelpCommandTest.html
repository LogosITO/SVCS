Подробное описание

Google Test фикстур для запуска модульных/интеграционных тестов HelpCommand.

Этот фикстур критически важен для изоляции логики HelpCommand, обеспечивая правильное делегирование запросов справки в MockHelpService и отчетность о результатах через MockSubject (шину событий).
