Подробное описание

Команда для создания центральных репозиториев-хабов для командного сотрудничества.

Реализует функциональность 'svcs hub' для инициализации голых репозиториев, которые действуют как центральные точки для отправки, получения и совместной работы над кодом. Эта команда необходима для настройки инфраструктуры удаленного сотрудничества.
