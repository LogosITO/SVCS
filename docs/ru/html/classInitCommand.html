Подробное описание

Class implementing the logic for the 'init' command.

This command is responsible for creating a new SVCS repository structure (.svcs) in the specified directory or the current one.

bool InitCommand::execute ( const std::vector< std::string > & args)
overridevirtual

Executes the repository initialization logic.

Determines the initialization path (from arguments or the current directory), checks for an existing repository, and creates a new one.

Аргументы
argsA vector of string arguments passed to the command. It's expected to have at most one argument—the target path.
Возвращает
true if the repository was successfully created; false in case of an error (e.g., already exists).
