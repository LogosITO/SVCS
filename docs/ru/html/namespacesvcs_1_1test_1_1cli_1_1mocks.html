Подробное описание

Mock объекты и тестовые дубли для тестирования CLI команд.

Содержит mock реализации CLI-интерфейсов, используемых для тестирования поведения команд, систем справки и взаимодействий с пользователем. Эти моки предоставляют контролируемые ответы команд и возможности верификации для модульных тестов CLI команд.
