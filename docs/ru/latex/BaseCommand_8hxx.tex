\doxylink{classBaseCommand}{Base\+Command} реализует логику взаимодействия с системой шины событий, позволяя всем производным командам легко генерировать информационные, отладочные и другие сообщения, которые будут автоматически обработаны зарегистрированными наблюдателями (например, \doxylink{classLogger}{Logger}).
