Эта команда действует как интерфейс командной строки для операций с ветками. Она разбирает аргументы командной строки для определения требуемого действия (список, создание, удаление, переименование, переключение или показ текущей) и вызывает соответствующий метод в основном \doxylink{classBranchManager}{Branch\+Manager}.
