Объявление класса Branch\+Command для управления ветками. 

\begin{DoxyCopyright}{Авторство}
Copyright 2025 Logos\+ITO Licensed under MIT-\/\+License
\end{DoxyCopyright}
Branch\+Command является интерфейсом для команды \textquotesingle{}svcs branch\textquotesingle{}. Он поддерживает различные операции с ветками, включая список существующих веток, создание, удаление, переименование и переключение веток. Он полагается на Branch\+Manager для основной логики и шину событий ISubject для общения с пользователем.
