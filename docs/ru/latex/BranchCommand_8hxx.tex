\doxylink{classBranchCommand}{Branch\+Command} является интерфейсом для команды \textquotesingle{}svcs branch\textquotesingle{}. Он поддерживает различные операции с ветками, включая список существующих веток, создание, удаление, переименование и переключение веток. Он полагается на \doxylink{classBranchManager}{Branch\+Manager} для основной логики и шину событий \doxylink{classISubject}{ISubject} для общения с пользователем.
