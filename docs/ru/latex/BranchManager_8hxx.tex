\doxysubsection{Подробное описание}
Declaration of the \doxylink{classBranchManager}{Branch\+Manager} class for handling version control branches. 

\doxylink{classBranchManager}{Branch\+Manager} отвечает за все операции, связанные с управлением ветвями в репозитории SVCS, включая создание, удаление, переименование и переключение. Он управляет метаданными ветвей в памяти и сохраняет изменения в файловой системе, используя служебные методы для загрузки и сохранения состояния.
