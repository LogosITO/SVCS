Hub\+Command реализует команду \textquotesingle{}svcs hub\textquotesingle{} для создания центральных голых репозиториев, которые служат центрами сотрудничества для команд разработчиков. Эти репозитории не имеют рабочих директорий и оптимизированы для удаленного доступа. Команда создает необходимую структуру директорий и файлы конфигурации, требуемые для функционального центрального репозитория.
