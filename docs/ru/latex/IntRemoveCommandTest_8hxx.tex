Объявляет фикстур Remove\+Command\+Test, который настраивает контролируемое окружение для тестирования команды \textquotesingle{}svcs remove\textquotesingle{} (или \textquotesingle{}svcs rm\textquotesingle{}). Этот фикстур управляет временным репозиторием SVCS и предоставляет вспомогательные функции для манипуляции файлами, индексации и проверки вывода.
