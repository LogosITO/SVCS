Этот файл предоставляет конкретную реализацию для фикстура Status\+Command\+Test. Он настраивает временный репозиторий SVCS с начальными неотслеживаемыми файлами и включает вспомогательные функции для создания файлов, директорий и индексации изменений. Фикстур предназначен для тестирования способности команды {\ttfamily svcs status} точно отчитываться о состоянии рабочего дерева.
