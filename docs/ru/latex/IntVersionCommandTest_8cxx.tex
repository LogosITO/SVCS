Этот файл предоставляет конкретную реализацию для фикстура Version\+Command\+Test. Поскольку команда {\ttfamily svcs version} не имеет состояния и не взаимодействует с репозиторием, настройка сосредоточена на инициализации Mock\+Subject (шины событий) для захвата вывода и самого экземпляра команды. Предоставлены вспомогательные методы для проверки точного содержимого уведомлений команды.
