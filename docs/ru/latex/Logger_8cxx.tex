Этот файл предоставляет конкретную реализацию для класса \doxylink{classLogger}{Logger}, включая управление шаблоном Singleton, фильтрацию уровней логов, форматирование сообщений и интеграцию с шиной событий. \doxylink{classLogger}{Logger} автоматически обрабатывает системные события из шины событий и предоставляет прямые методы логирования для ручного использования.
