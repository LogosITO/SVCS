\doxysubsection{Подробное описание}
Mock реализация интерфейса \doxylink{classHelpService}{Help\+Service} для целей модульного тестирования. 

\begin{DoxyCopyright}{Авторство}
Copyright 2025 Logos\+ITO Licensed under MIT-\/\+License
\end{DoxyCopyright}
Этот класс наследуется от \doxylink{classHelpService}{Help\+Service} и переопределяет его абстрактные методы ({\ttfamily get\+Available\+Commands}, {\ttfamily get\+Command\+Description}, {\ttfamily show\+Command\+Help}) контролируемыми реализациями. Он включает служебные методы ({\ttfamily set\+Available\+Commands}, {\ttfamily was\+Command\+Help\+Called} и т.\+д.) чтобы позволить тестам настраивать поведение mock\textquotesingle{}а и проверять его взаимодействия. Этот mock необходим для модульного тестирования {\bfseries{\doxylink{classHelpCommand}{Help\+Command}}} без зависимости от фактической реализации системы регистрации CLI команд.
