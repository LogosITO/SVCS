\doxysubsection{Подробное описание}
Mock реализация интерфейса Help\+Service для целей модульного тестирования. 

\begin{DoxyCopyright}{Авторство}
Copyright 2025 Logos\+ITO Licensed under MIT-\/\+License
\end{DoxyCopyright}
Этот класс предоставляет mock реализацию с тем же интерфейсом, что и Help\+Service, но с контролируемым поведением для тестирования. Он включает служебные методы ({\ttfamily set\+Available\+Commands}, {\ttfamily was\+Command\+Help\+Called} и т.\+д.) чтобы позволить тестам настраивать поведение mock\textquotesingle{}а и проверять его взаимодействия. Этот mock необходим для модульного тестирования {\bfseries{Help\+Command}} без зависимости от фактической реализации системы регистрации CLI команд.
