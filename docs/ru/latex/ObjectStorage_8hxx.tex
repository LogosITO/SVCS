\doxysubsection{Подробное описание}
Управляет чтением, записью и жизненным циклом объектов СКВ на диске с использованием формата Git. 

\begin{DoxyCopyright}{Авторство}
Copyright 2025 Logos\+ITO Licensed under MIT-\/\+License
\end{DoxyCopyright}
Этот класс обрабатывает построение путей к файлам, сериализацию, сжатие/распаковку Zlib и действует как фабрика для восстановления конкретных типов Vcs\+Object (Blob, Tree, Commit). Использует внедренный ISubject для внутреннего логирования событий.
