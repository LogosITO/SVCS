\doxylink{classRemoteProtocol}{Remote\+Protocol} реализует пользовательский протокол для операций push/pull между клиентами и серверами SVCS. Он обрабатывает согласование объектов, передачу и обновления ссылок через простой текстовый протокол по stdin/stdout.
