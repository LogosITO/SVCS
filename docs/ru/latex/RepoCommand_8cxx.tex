Реализация класса Repo\+Command для управления конфигурациями удаленных репозиториев.

Этот файл реализует конкретную логику для команды "{}svcs repo"{}. Он в основном делегирует свою основную функциональность—добавление, удаление, переименование и перечисление удаленных репозиториев— классу Remote\+Manager. Реализация включает\+:
\begin{DoxyEnumerate}
\item {\bfseries{Проверку контекста репозитория}}\+: Обеспечивает выполнение команды внутри инициализированного репозитория SVCS.
\item {\bfseries{Диспетчеризацию подкоманд}}\+: Разбирает первый аргумент для определения, какой обработчик ({\ttfamily handle\+Add}, {\ttfamily handle\+Remove} и т.\+д.) вызвать.
\item {\bfseries{Проверку аргументов}}\+: Выполняет проверки количества аргументов для каждой подкоманды.
\item {\bfseries{Вывод пользователю}}\+: Напрямую выводит сообщения подтверждения ({\ttfamily std\+::cout}) или использует шину событий ISubject для ошибок и информационных сообщений. 
\end{DoxyEnumerate}
