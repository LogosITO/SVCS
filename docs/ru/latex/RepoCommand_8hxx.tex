\doxylink{classRepoCommand}{Repo\+Command} реализует интерфейс \doxylink{classICommand}{ICommand} для управления конфигурациями на уровне репозитория, в основном сосредотачиваясь на управлении {\bfseries{удаленными репозиториями}}. Эта команда действует как фасад, делегируя фактическое управление удаленными репозиториями классу \doxylink{classRemoteManager}{Remote\+Manager} после обеспечения выполнения команды в контексте валидного репозитория (неявно обрабатывается \doxylink{classRepositoryManager}{Repository\+Manager}). Команда поддерживает подкоманды такие как {\ttfamily add}, {\ttfamily remove}, {\ttfamily list} и {\ttfamily rename}. Наследуется от \doxylink{classServerBaseCommand}{Server\+Base\+Command} для общей функциональности серверных команд.
