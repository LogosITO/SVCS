Repo\+Command реализует интерфейс ICommand для управления конфигурациями на уровне репозитория, в основном сосредотачиваясь на управлении {\bfseries{удаленными репозиториями}}. Эта команда действует как фасад, делегируя фактическое управление удаленными репозиториями классу Remote\+Manager после обеспечения выполнения команды в контексте валидного репозитория (неявно обрабатывается Repository\+Manager). Команда поддерживает подкоманды такие как {\ttfamily add}, {\ttfamily remove}, {\ttfamily list} и {\ttfamily rename}. Наследуется от Server\+Base\+Command для общей функциональности серверных команд.
