\doxysubsection{Подробное описание}
Управляет всем репозиторием системы контроля версий, действует как Subject для логирования событий и контролирует все основные операции СКВ.

Этот класс отвечает за инициализацию структуры репозитория, управление \doxylink{classObjectStorage}{Object\+Storage} и координацию всех взаимодействий с файловой системой и базой данных объектов.
