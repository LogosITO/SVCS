std\+::string read\+\_\+file\+\_\+to\+\_\+string (\begin{DoxyParamCaption}\item[{const fs\+::path \&}]{full\+\_\+path}{}\end{DoxyParamCaption})}



Читает всё содержимое файла в одну строку std\+::string. 

Эта служебная функция предназначена для чтения файлов малого и среднего размера и является критически важной для создания содержимого \doxylink{classVcsObject}{Vcs\+Object} (например, Blobs). Она открывает файл в бинарном режиме, чтобы гарантировать корректное чтение всех символов на всех платформах. 
\begin{DoxyParams}{Аргументы}
{\em full\+\_\+path} & Полный, абсолютный путь к файлу на диске. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Возвращает}
Всё содержимое файла в виде одной строки std\+::string. 
\end{DoxyReturn}

\begin{DoxyExceptions}{Исключения}
{\em std\+::runtime\+\_\+error} & Если файл не может быть открыт для чтения. \\
\hline
\end{DoxyExceptions}
