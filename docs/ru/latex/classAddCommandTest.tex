\doxysubsection{Подробное описание}
Тестовый фикстур для интеграционных тестов \doxylink{classAddCommand}{Add\+Command}. 

Управляет настройкой и очисткой временного инициализированного репозитория SVCS. Предоставляет служебные методы для создания различных структур файловой системы для тестирования логики добавления в индекс команды \doxylink{classAddCommand}{Add\+Command}.
