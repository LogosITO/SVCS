\doxysubsection{Подробное описание}
Базовый класс для всех CLI команд. 

Предоставляет общую инфраструктуру для взаимодействия с системой шины событий (\doxylink{classISubject}{ISubject}) и служебные методы для работы с репозиторием. Отвечает за логирование через шину событий.
