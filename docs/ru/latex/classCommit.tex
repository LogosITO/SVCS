\doxysubsection{Подробное описание}
Представляет единую, неизменяемую историческую точку (снимок) в истории репозитория. 

Этот объект связывает структуру содержимого (хеш \doxylink{classTree}{Tree}) с метаданными (автор, сообщение, история родителей).
