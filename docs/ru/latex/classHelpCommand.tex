\doxysubsection{Подробное описание}
Команда для отображения справочной информации о других командах. 

{\bfseries{\doxylink{classHelpCommand}{Help\+Command}}} является встроенной утилитой справки приложения. Она может показывать общий список всех доступных команд или отображать подробное использование и описания для конкретной команды, предоставленной в качестве аргумента. Она делегирует получение информации о командах в {\bfseries{\doxylink{classHelpService}{Help\+Service}}} для поддержания чистого разделения ответственности.
