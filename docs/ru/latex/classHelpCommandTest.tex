\doxysubsection{Подробное описание}
Google Test фикстур для запуска модульных/интеграционных тестов \doxylink{classHelpCommand}{Help\+Command}. 

Этот фикстур критически важен для изоляции логики \doxylink{classHelpCommand}{Help\+Command}, обеспечивая правильное делегирование запросов справки в {\bfseries{\doxylink{classMockHelpService}{Mock\+Help\+Service}}} и отчетность о результатах через {\bfseries{\doxylink{classMockSubject}{Mock\+Subject}}} (шину событий).
