\doxysubsection{Подробное описание}
Сервис, предоставляющий справочную информацию для команд. 

Этот сервис реализует логику приложения для отображения справки. Он использует набор {\bfseries{функций обратного вызова}} ({\ttfamily std\+::function}) для получения данных команд (имена, описания, использование) и отображения конкретной справки. Этот выбор дизайна предотвращает прямую зависимость от \doxylink{classCommandFactory}{Command\+Factory}, тем самым разрывая потенциальную {\bfseries{циклическую зависимость}}.
