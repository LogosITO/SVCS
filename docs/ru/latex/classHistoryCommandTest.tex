\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classHistoryCommand}{History\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для интеграционных тестов команды history\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classMockSubject}{Mock\+Subject} (для шины событий) и \doxylink{classRepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают создание файлов, индексацию и, что важно, создание mock коммитов для заполнения истории для тестирования различных форматов вывода и ограничений. 
\end{DoxyItemize}
