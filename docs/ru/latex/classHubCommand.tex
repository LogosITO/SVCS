\doxysubsection{Подробное описание}
Команда для создания центральных репозиториев-\/хабов для командного сотрудничества. 

Реализует функциональность \textquotesingle{}svcs hub\textquotesingle{} для инициализации голых репозиториев, которые действуют как центральные точки для отправки, получения и совместной работы над кодом. Эта команда необходима для настройки инфраструктуры удаленного сотрудничества. Наследуется от \doxylink{classServerBaseCommand}{Server\+Base\+Command} для общей функциональности серверных команд.
