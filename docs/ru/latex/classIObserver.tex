\doxysubsection{Подробное описание}
Интерфейс Observer (Абстрактный Базовый Класс). 

Любой класс, который хочет получать уведомления от ядра системы (например, \doxylink{classRepository}{Repository}, Command), должен реализовать этот интерфейс для регистрации в Subject.
