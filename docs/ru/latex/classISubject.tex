\doxysubsection{Подробное описание}
Интерфейс Subject (Издатель) (Абстрактный Базовый Класс). 

Основные классы (например, \doxylink{classRepository}{Repository}, \doxylink{classEventBus}{Event\+Bus}) реализуют этот интерфейс, чтобы позволить наблюдателям подписываться на события. Контракт требует, чтобы все функции управления ({\ttfamily attach}, {\ttfamily detach}) обрабатывали {\ttfamily std\+::shared\+\_\+ptr\texorpdfstring{$<$}{<}\doxylink{classIObserver}{IObserver}\texorpdfstring{$>$}{>}}.

Уведомляет всех прикрепленных наблюдателей о событии. 

Этот метод сделан публичным, чтобы классы компонентов (как \doxylink{classObjectStorage}{Object\+Storage}) могли публиковать события через внедренный указатель \doxylink{classISubject}{ISubject}. Метод объявлен как {\ttfamily const}, потому что уведомление о событии является логически неизменяющим действием над состоянием Subject.
