\doxysubsection{Подробное описание}
Реализует команду "{}init"{} для системы контроля версий (VCS). 

Эта команда отвечает за инициализацию нового репозитория SVCS в текущей директории или указанном местоположении, настраивая необходимую внутреннюю структуру (например, скрытую директорию .svcs). Класс наследуется от \doxylink{classICommand}{ICommand}, обеспечивая соответствие интерфейсу шаблона Command. Он требует шину событий для связи и менеджер репозитория для выполнения фактической логики инициализации.
