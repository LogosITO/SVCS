\doxysubsection{Подробное описание}
Тестовый фикстур для интеграционных тестов \doxylink{classInitCommand}{Init\+Command}. 

Управляет настройкой и очисткой временной изолированной директории в файловой системе для каждого тестового случая. Использует реальный \doxylink{classRepositoryManager}{Repository\+Manager} для тестирования фактической логики создания репозитория и \doxylink{classMockSubject}{Mock\+Subject} для проверки публикации событий.
