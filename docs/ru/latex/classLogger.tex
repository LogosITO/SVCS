\doxysubsection{Подробное описание}
Потокобезопасный Singleton класс, отвечающий за логирование системных событий. 

Реализует интерфейс \doxylink{classIObserver}{IObserver} для автоматического получения и обработки событий, сгенерированных командами и основными компонентами.
