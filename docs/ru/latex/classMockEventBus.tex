Уведомляет всех прикрепленных наблюдателей о событии. 

Этот метод сделан публичным, чтобы классы компонентов (как \doxylink{classObjectStorage}{Object\+Storage}) могли публиковать события через внедренный указатель \doxylink{classISubject}{ISubject}. Метод объявлен как {\ttfamily const}, потому что уведомление о событии является логически неизменяющим действием над состоянием Subject.
