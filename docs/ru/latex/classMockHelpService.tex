\doxysubsection{Подробное описание}
Mock реализация \doxylink{classHelpService}{Help\+Service} для тестирования \doxylink{classHelpCommand}{Help\+Command}. 

Этот mock позволяет тестовым случаям\+:
\begin{DoxyEnumerate}
\item {\bfseries{Настраивать}} список доступных команд.
\item {\bfseries{Настраивать}} описания для конкретных команд.
\item {\bfseries{Проверять}}, справка какой команды была запрошена {\ttfamily \doxylink{classHelpCommand}{Help\+Command}}.
\item Использовать предоставленный \doxylink{classMockSubject}{Mock\+Subject} для вывода отладочной информации во время тестирования. 
\end{DoxyEnumerate}
