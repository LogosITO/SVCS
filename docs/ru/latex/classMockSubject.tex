\doxysubsection{Подробное описание}
Mock implementation of \doxylink{classISubject}{ISubject} (\doxylink{structEvent}{Event} Bus). 

Mock реализация \doxylink{classISubject}{ISubject} для тестирования событийно-\/ориентированных компонентов.

Mock implementation of \doxylink{classISubject}{ISubject}.


\begin{DoxyItemize}
\item This mock silences all notifications, ensuring that \doxylink{classIndex}{Index} logic is tested without side effects from the event system.
\item This mock silences all event notifications, ensuring \doxylink{classObjectStorage}{Object\+Storage} tests run without side effects from the event system.

Этот mock позволяет тестовым случаям\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item {\bfseries{Захватывать}} все события, опубликованные через метод \doxylink{classMockSubject_a9aaffe9cd012979c90c186ea3ec86fc0}{notify()}.
\item {\bfseries{Проверять}} захваченные события на наличие определенных типов или сообщений.
\item {\bfseries{Очищать}} историю событий между тестовыми случаями.
\item {\bfseries{Проверять}}, что определенные события были опубликованы во время выполнения. 
\end{DoxyEnumerate}
