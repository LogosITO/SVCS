\doxysubsection{Подробное описание}
Управляет конфигурациями удаленных репозиториев. 

Этот класс предоставляет интерфейс для управления удаленными репозиториями. Он обрабатывает добавление, удаление и перечисление удаленных репозиториев, а также сохранение этих конфигураций в файл {\ttfamily .svcs/config} репозитория и загрузку из него.
