\doxysubsection{Подробное описание}
Обрабатывает протокол удаленной связи для SVCS. 

Управляет пользовательским протоколом для операций {\bfseries{push}} и {\bfseries{pull}}, включая согласование объектов, передачу данных и обновления ссылок между клиентом и сервером. Протокол представляет собой простой текстовый обмен через стандартный ввод/вывод.
