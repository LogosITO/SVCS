\doxysubsection{Подробное описание}
Команда для удаления файлов из области подготовки. 

\doxylink{classRemoveCommand}{Remove\+Command} удаляет файлы из области подготовки (индекса), эффективно убирая их из подготовки, сохраняя при этом сами файлы нетронутыми.
