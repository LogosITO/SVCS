\doxysubsection{Подробное описание}
Обработчик команд для управления удаленными репозиториями и конфигурациями. 

Реализует команду "{}svcs repo"{}, предоставляя подкоманды для управления списком удаленных репозиториев, связанных с текущим проектом. Наследуется от \doxylink{classServerBaseCommand}{Server\+Base\+Command} для общей функциональности серверных команд.
