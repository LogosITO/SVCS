\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classSaveCommand}{Save\+Command} (коммит). 

Этот фикстур предоставляет необходимую инфраструктуру для тестирования команды save\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classMockSubject}{Mock\+Subject} (для шины событий) и \doxylink{classRepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают создание файлов, настройку директорий, индексацию и проверку уведомлений. 
\end{DoxyItemize}
