\doxysubsection{Подробное описание}
Фабрика для создания объектов \doxylink{classICommand}{ICommand} администрирования сервера. 

\doxylink{classServerCommandFactory}{Server\+Command\+Factory} управляет жизненным циклом серверных команд, создавая их динамически с использованием функций-\/создателей, зарегистрированных под уникальными именами. Команды конструируются с общим владением шиной событий (\doxylink{classISubject}{ISubject}) и менеджером репозитория (\doxylink{classRepositoryManager}{Repository\+Manager}). Эта фабрика специально разработана для команд администрирования сервера, требующих управления репозиториями.
