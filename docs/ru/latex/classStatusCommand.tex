\doxysubsection{Подробное описание}
Команда для отображения текущего статуса репозитория. 

\doxylink{classStatusCommand}{Status\+Command} отображает\+:
\begin{DoxyItemize}
\item Текущую ветку
\item Подготовленные изменения (готовые к сохранению)
\item Неподготовленные изменения (еще не добавленные)
\item Неотслеживаемые файлы (новые файлы)
\item Статус конкретных файлов при предоставлении в качестве аргументов 
\end{DoxyItemize}
