\doxysubsection{Подробное описание}
Вспомогательный класс для тестирования неабстрактных методов \doxylink{classVcsObject}{Vcs\+Object}. 

Этот класс реализует все чисто виртуальные методы \doxylink{classVcsObject}{Vcs\+Object} простыми заглушками, позволяя создавать его экземпляры для модульного тестирования логики хеширования в \doxylink{classVcsObject_ae2b7544a63a2f0b1ef764dd4710a13e0}{Vcs\+Object\+::compute\+Hash}.
