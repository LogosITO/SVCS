\doxysubsection{Подробное описание}
Команда для возврата состояния репозитория к предыдущему коммиту. 

Реализует функциональность \textquotesingle{}svcs undo\textquotesingle{}. Он использует \doxylink{classRepositoryManager}{Repository\+Manager} для выполнения фактических манипуляций с историей и рабочей директорией. Он использует \doxylink{classISubject}{ISubject} (шину событий) для вывода пользователю и запросов подтверждения.
