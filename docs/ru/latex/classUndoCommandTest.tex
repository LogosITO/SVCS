\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classUndoCommand}{Undo\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для тестирования команды undo\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classMockSubject}{Mock\+Subject} (для шины событий/вывода/имитации ввода) и \doxylink{classRepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают манипуляцию файлами, индексацию, коммиты и проверку уведомлений. 
\end{DoxyItemize}
