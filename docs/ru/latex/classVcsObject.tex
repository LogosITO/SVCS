\doxysubsection{Подробное описание}
Абстрактный базовый класс, представляющий любой сохраняемый, адресуемый объект в СКВ. 

\doxylink{classVcsObject}{Vcs\+Object} обеспечивает обязательное наличие хеш-\/идентификатора, логики сериализации и идентификации типа для всех производных классов. Он служит основной доменной сущностью для базы данных объектов.
