\doxysubsection{Подробное описание}
Google Test фикстур для запуска модульных/интеграционных тестов \doxylink{classVersionCommand}{Version\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для тестирования \doxylink{classVersionCommand}{Version\+Command}\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Инициализация и очистка mock шины событий и объекта команды.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classMockSubject}{Mock\+Subject} (для захвата вывода) и самого \doxylink{classVersionCommand}{Version\+Command}.
\item {\bfseries{Вспомогательные методы\+:}} Упрощают проверку захваченных уведомлений о событиях на наличие конкретного содержимого. 
\end{DoxyItemize}
