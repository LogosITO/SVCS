\doxysubsection{Подробное описание}
Фабрика для создания объектов \doxylink{classsvcs_1_1cli_1_1ICommand}{ICommand} по их строковому имени. 

\doxylink{classsvcs_1_1cli_1_1CommandFactory}{Command\+Factory} управляет жизненным циклом команд, создавая их динамически с использованием функций-\/создателей, зарегистрированных под уникальными именами. Команды конструируются с общим владением шиной событий (\doxylink{classsvcs_1_1services_1_1ISubject}{ISubject}) и менеджером репозитория (\doxylink{classsvcs_1_1core_1_1RepositoryManager}{Repository\+Manager}).
