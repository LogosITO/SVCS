\doxysubsection{Подробное описание}
Сервис, предоставляющий справочную информацию для команд. 

Этот сервис реализует логику приложения для отображения справки. Он использует набор {\bfseries{функций обратного вызова}} ({\ttfamily std\+::function}) для получения данных команд (имена, описания, использование) и отображения конкретной справки. Этот выбор дизайна предотвращает прямую зависимость от \doxylink{classsvcs_1_1cli_1_1CommandFactory}{Command\+Factory}, тем самым разрывая потенциальную {\bfseries{циклическую зависимость}}.
