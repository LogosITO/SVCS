\doxysubsection{Подробное описание}
Команда для возврата состояния репозитория к предыдущему коммиту. 

Реализует функциональность \textquotesingle{}svcs undo\textquotesingle{}. Он использует \doxylink{classsvcs_1_1core_1_1RepositoryManager}{Repository\+Manager} для выполнения фактических манипуляций с историей и рабочей директорией. Он использует \doxylink{classsvcs_1_1services_1_1ISubject}{ISubject} (шину событий) для вывода пользователю и запросов подтверждения.
