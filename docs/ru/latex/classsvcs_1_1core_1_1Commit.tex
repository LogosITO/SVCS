\doxysubsection{Подробное описание}
Представляет единую, неизменяемую историческую точку (снимок) в истории репозитория. 

Этот объект связывает структуру содержимого (хеш \doxylink{classsvcs_1_1core_1_1Tree}{Tree}) с метаданными (автор, сообщение, история родителей).
