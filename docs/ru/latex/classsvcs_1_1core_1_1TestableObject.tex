\doxysubsection{Подробное описание}
Вспомогательный класс для тестирования неабстрактных методов \doxylink{classsvcs_1_1core_1_1VcsObject}{Vcs\+Object}. 

Этот класс реализует все чисто виртуальные методы \doxylink{classsvcs_1_1core_1_1VcsObject}{Vcs\+Object} простыми заглушками, позволяя создавать его экземпляры для модульного тестирования логики хеширования в \doxylink{classsvcs_1_1core_1_1VcsObject_a776323941cf6d9f2923b67cf9cc95ec5}{Vcs\+Object\+::compute\+Hash}.
