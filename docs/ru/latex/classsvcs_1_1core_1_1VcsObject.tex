\doxysubsection{Подробное описание}
Абстрактный базовый класс, представляющий любой сохраняемый, адресуемый объект в СКВ. 

\doxylink{classsvcs_1_1core_1_1VcsObject}{Vcs\+Object} обеспечивает обязательное наличие хеш-\/идентификатора, логики сериализации и идентификации типа для всех производных классов. Он служит основной доменной сущностью для базы данных объектов.
