\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classsvcs_1_1cli_1_1StatusCommand}{Status\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для интеграционных тестов\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classsvcs_1_1test_1_1cli_1_1mocks_1_1MockSubject}{Mock\+Subject} (для шины событий) и \doxylink{classsvcs_1_1core_1_1RepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают создание файлов, настройку директорий, индексацию и проверку уведомлений. 
\end{DoxyItemize}
