\doxysubsection{Подробное описание}
Google Test фикстур для запуска модульных/интеграционных тестов \doxylink{classsvcs_1_1cli_1_1VersionCommand}{Version\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для тестирования \doxylink{classsvcs_1_1cli_1_1VersionCommand}{Version\+Command}\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Инициализация и очистка mock шины событий и объекта команды.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classsvcs_1_1test_1_1cli_1_1mocks_1_1MockSubject}{Mock\+Subject} (для захвата вывода) и самого \doxylink{classsvcs_1_1cli_1_1VersionCommand}{Version\+Command}.
\item {\bfseries{Вспомогательные методы\+:}} Упрощают проверку захваченных уведомлений о событиях на наличие конкретного содержимого. 
\end{DoxyItemize}
