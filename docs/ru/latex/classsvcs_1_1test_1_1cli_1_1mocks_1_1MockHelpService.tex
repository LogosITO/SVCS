\doxysubsection{Подробное описание}
Mock реализация \doxylink{classsvcs_1_1cli_1_1HelpService}{Help\+Service} для тестирования \doxylink{classsvcs_1_1test_1_1cli_1_1HelpCommand}{Help\+Command}. 

Этот mock позволяет тестовым случаям\+:
\begin{DoxyEnumerate}
\item {\bfseries{Настраивать}} список доступных команд.
\item {\bfseries{Настраивать}} описания для конкретных команд.
\item {\bfseries{Проверять}}, справка какой команды была запрошена {\ttfamily \doxylink{classsvcs_1_1test_1_1cli_1_1HelpCommand}{Help\+Command}}.
\item Использовать предоставленный \doxylink{classsvcs_1_1test_1_1cli_1_1mocks_1_1MockSubject}{Mock\+Subject} для вывода отладочной информации во время тестирования. 
\end{DoxyEnumerate}
