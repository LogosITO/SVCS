\doxysubsection{Подробное описание}
Mock реализация \doxylink{classsvcs_1_1services_1_1ISubject}{ISubject} для тестирования событийно-\/ориентированных компонентов. 

Этот mock позволяет тестовым случаям\+:
\begin{DoxyEnumerate}
\item {\bfseries{Захватывать}} все события, опубликованные через метод \doxylink{classsvcs_1_1test_1_1cli_1_1mocks_1_1MockSubject_abfc5bfa7fe007eca3e7e8dd8d84e1d9c}{notify()}.
\item {\bfseries{Проверять}} захваченные события на наличие определенных типов или сообщений.
\item {\bfseries{Очищать}} историю событий между тестовыми случаями.
\item {\bfseries{Проверять}}, что определенные события были опубликованы во время выполнения. 
\end{DoxyEnumerate}
