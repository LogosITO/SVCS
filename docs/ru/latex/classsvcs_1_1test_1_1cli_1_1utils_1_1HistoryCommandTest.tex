\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1HistoryCommand}{History\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для интеграционных тестов команды history\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1MockSubject}{Mock\+Subject} (для шины событий) и \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1RepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают создание файлов, индексацию и, что важно, создание mock коммитов для заполнения истории для тестирования различных форматов вывода и ограничений. 
\end{DoxyItemize}
