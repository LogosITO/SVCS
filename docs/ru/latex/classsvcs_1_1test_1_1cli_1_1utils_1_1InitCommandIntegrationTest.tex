Тестовый фикстур для интеграционных тестов \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1InitCommand}{Init\+Command}. 

Управляет настройкой и очисткой временной изолированной директории в файловой системе для каждого тестового случая. Использует реальный \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1RepositoryManager}{Repository\+Manager} для тестирования фактической логики создания репозитория и \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1MockSubject}{Mock\+Subject} для проверки публикации событий.
