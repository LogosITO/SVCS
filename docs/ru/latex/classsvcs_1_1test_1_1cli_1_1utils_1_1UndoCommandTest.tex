\doxysubsection{Подробное описание}
Google Test фикстур для запуска интеграционных тестов \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1UndoCommand}{Undo\+Command}. 

Этот фикстур предоставляет необходимую инфраструктуру для тестирования команды undo\+:
\begin{DoxyItemize}
\item {\bfseries{Set\+Up/\+Tear\+Down\+:}} Управление временной директорией репозитория SVCS.
\item {\bfseries{Основные компоненты\+:}} Экземпляры \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1MockSubject}{Mock\+Subject} (для шины событий/вывода/имитации ввода) и \doxylink{classsvcs_1_1test_1_1cli_1_1utils_1_1RepositoryManager}{Repository\+Manager} (для основной логики).
\item {\bfseries{Вспомогательные методы\+:}} Упрощают манипуляцию файлами, индексацию, коммиты и проверку уведомлений. 
\end{DoxyItemize}
