Уведомляет всех прикрепленных наблюдателей о событии. 

Этот метод сделан публичным, чтобы классы компонентов (как \doxylink{classsvcs_1_1core_1_1ObjectStorage}{Object\+Storage}) могли публиковать события через внедренный указатель \doxylink{classsvcs_1_1services_1_1ISubject}{ISubject}. Метод объявлен как {\ttfamily const}, потому что уведомление о событии является логически неизменяющим действием над состоянием Subject.
