{\bfseries{SVCS (Simple Version Control System)}} is an educational and research project aimed at implementing the fundamental principles of a distributed version control system (similar to Git) from scratch using C++.

The main goal of the project is to gain a deep understanding of core concepts\+:


\begin{DoxyItemize}
\item {\bfseries{Immutable Objects\+:}} Creating and handling objects ({\ttfamily \doxylink{classCommit}{Commit}}, {\ttfamily \doxylink{classBlob}{Blob}}, {\ttfamily \doxylink{classTree}{Tree}}) based on a single abstract {\ttfamily \doxylink{classVcsObject}{Vcs\+Object}} class. ~\newline

\item {\bfseries{Data Integrity\+:}} Applying cryptographic hashing ({\bfseries{SHA-\/256}}) for data identification and verification. ~\newline

\item {\bfseries{Clean Architecture\+:}} Implementing OOP principles, inheritance, polymorphism, and separation of concerns. ~\newline

\end{DoxyItemize}
