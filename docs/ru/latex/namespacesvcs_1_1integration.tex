\doxysubsection{Подробное описание}
Компоненты интеграции системы и сквозные задачи. 

Содержит компоненты, которые интегрируют различные подсистемы SVCS, включая интеграцию фабрик команд, оркестрацию сервисов и межслоевое взаимодействие. Это пространство имен служит мостом между CLI, сервером, ядром и сервисными слоями для обеспечения единого пользовательского опыта приложения.
