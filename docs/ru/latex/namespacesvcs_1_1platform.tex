\doxysubsection{Подробное описание}
Platform abstraction layer. 

This namespace contains cross-\/platform wrappers for operating system functionality, allowing the code to work seamlessly on Windows, Linux, mac\+OS, and other POSIX systems.

Слой абстракции платформы Это пространство имен содержит кросс-\/платформенные обертки для функциональности операционной системы, позволяя коду работать без изменений на Windows, Linux, mac\+OS и других POSIX-\/системах.
