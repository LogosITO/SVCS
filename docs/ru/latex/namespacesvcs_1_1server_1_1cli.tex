\doxysubsection{Подробное описание}
Компоненты командной строки для администрирования сервера. 

Содержит CLI команды, специально предназначенные для администрирования и управления сервером, включая создание репозиториев-\/хабов, управление пользователями и конфигурацию сервера. Эти команды используются администраторами для настройки и обслуживания экземпляров сервера SVCS. Это пространство имен связывает серверную функциональность с интерфейсом командной строки.
