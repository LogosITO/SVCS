\doxysubsection{Подробное описание}
Служебные классы и тестовые фикстуры для тестирования CLI команд. 

Содержит тестовые фикстуры, вспомогательные классы и служебные функции, которые поддерживают тестирование CLI команд. Эти компоненты предоставляют общие шаблоны настройки, очистки и тестирования для интеграционных тестов команд.
