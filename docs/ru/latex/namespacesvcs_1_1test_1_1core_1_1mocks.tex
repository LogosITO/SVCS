Mock объекты и тестовые дубли для тестирования основных компонентов. 

Содержит mock реализации основных интерфейсов, используемых для изоляции и тестирования основных компонентов СКВ. Эти моки предоставляют контролируемое поведение и возможности верификации для модульных тестов.
