\doxysubsection{Подробное описание}
Представляет одну запись файла в области подготовки (\doxylink{classIndex}{Index}). 

\doxylink{structIndexEntry}{Index\+Entry} хранит информацию о файле, который подготовлен для следующего коммита, включая его ID объекта (хеш) и метаданные файла, используемые для обнаружения изменений.
