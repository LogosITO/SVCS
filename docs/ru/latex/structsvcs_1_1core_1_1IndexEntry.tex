\doxysubsection{Подробное описание}
Представляет одну запись файла в области подготовки (\doxylink{classsvcs_1_1core_1_1Index}{Index}). 

\doxylink{structsvcs_1_1core_1_1IndexEntry}{Index\+Entry} хранит информацию о файле, который подготовлен для следующего коммита, включая его ID объекта (хеш) и метаданные файла, используемые для обнаружения изменений.
